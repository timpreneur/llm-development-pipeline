# pipeline-substrate — lessons

Append dated entries below when the substrate convention itself reveals a gap — a directory rule that doesn't survive real features, a `_meta/` shape that LLM Ops can't parse, a frontmatter flag that needs a new vocabulary value, etc.

Format:

```markdown
## YYYY-MM-DD — <short title>
<one paragraph: what happened, why it's a pattern not a one-off, what the suggested convention change is>
```

`pipeline-llm-ops` reconciles pending entries — promotes, retires, or discards.
